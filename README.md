# Jackson Housing Connection

## Purpose

This website helps connect people seeking housing in Jackson with community members who have ADUs or small rentals available. All matching is done manually by the Jackson Housing Commission, and housing is not guaranteed.

## How to Use

- If you need housing, click the link and fill out the Google Form.
- If you have housing to offer, click the link and fill out the Google Form.
- The Housing Commission reviews all submissions and follows up directly.

## Tech Stack

- Static HTML pages
- Google Forms + Google Sheets
- Hosted on GitHub Pages
